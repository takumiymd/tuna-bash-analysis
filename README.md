# Tokyo Wholesale Tuna Analysis (Bash)

This project analyzes a tuna dataset using basic bash UNIX tools

## Features
- Counts and grouping
- Price and quantity statistics
- Top values and yearly records

## Dataset
Tokyo Wholesale Tuna Prices  
Source: Kaggle  
License: Creative Commons Attribution 4.0 (CC BY 4.0)

Link: https://www.kaggle.com/datasets/tcashion/tokyo-wholesale-tuna-prices/data
